Theme Name:         Alexis
Theme URI:          https://wpqa.net/demo/
Description:        Alexis is a mobile responsive, flat design WordPress theme with a touch of minimalism. It comes with an advanced customizer that 					allows you to change theme color and other aspects without touching a single line of code.
Version:            1.4.0
Author:             Yan Susanto
Author URI:         http://www.yansusanto.com


== Features ==

[1] Compatible with the latest WordPress version
[2] 100% responsive design
[3] 3-level dropdown menu
[4] Widget-ready sidebar
[5] Full width page template
[6] Threaded comments
[7] Support for Contact Form 7
[8] Compatible with all major browsers


== Demo ==
https://wpqa.net/demo/


== Use ==

[1] This theme can be used right out of the box but if you, for some reasons, happen to dislike the theme color, you can easily change it via the theme 	options in the Customize section of the Dashboard under Appearance > Customize > Theme Options. 
[2] If you haven’t created a custom menu yet, go to Appearance > Menus > create a new menu. 
	[2.1] Give it a name, then click [Create Menu]. 
	[2.2] Add your menu items
	[2.3] Assign it to 'Primary Navigation' under the 'Menu Settings'
	[2.4] Click [Save Menu]
